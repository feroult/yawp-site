Google App Engine Deploy

After you’ve bootstrapped your API with the aid of YAWP!’s Maven archetype, it is really straightforward to deploy it to Google App Engine. Just follow these steps:

1) Create your App Engine Project ID using Google Cloud Console.

2) Change the file src/main/webapp/WEB-IF/appengine-web.xml and insert your Project ID:

<application>YOUR_APPENGINE_PROJECT_ID</application>

3) From the project’s root folder, deploy your app using the App Engine Maven plugin:

mvn appengine:update

4) Test it with cURL:

$ curl http://YOUR_APPENGINE_PROJECT_ID.appspot.com/api
